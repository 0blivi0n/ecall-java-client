mercury-java-client
===================

##Introduction

  Mercury's JAVA client is a generic driver for invocation of mercury's RPC servers.
  
  
##Instalation

Maven dependency:
 
 ```xml
<dependency>
    <groupId>org.oblivion-cache</groupId>
    <artifactId>mercury-java-client</artifactId>
    <version>0.5.1</version>
</dependency>
 ```
